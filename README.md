#Project Overview
This project analyzes the question “Who wins the Oscars?” with a focus on how demographic factors influence the likelihood of winning an Academy Award in major categories.

The analysis examines patterns, representation, and changes over time among Oscar winners and nominees for Actors and Directors.

#Research Questions
- How do age, gender, race, and other demographics affect Oscar wins?
- Are certain demographic groups underrepresented or more successful?
- Has inclusivity evolved over time in major Oscar categories?
- How does the general public react to Oscar-winning movies featuring actors/actresses from minority groups?

#Data Preparation & Transformation
- Subsetting demographic data
- Handling missing values
- Data transformation and cleaning
- Subsetting movie information
